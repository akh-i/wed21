Jenkins Installation on Linux:-
-----------------------------------------
~create an ubuntu instance and install jdk,jenkins on it
1. sudo su -
2. sudo apt-get update
3. sudo apt-get install default-jre 
4. apt-get install default-jdk -y
5. java -version
6. cd /tmp 
7. curl -fsSL https://pkg.jenkins.io/debian/jenkins.io.key | sudo tee /usr/share/keyrings/jenkins-keyring.asc > /dev/null
8. echo "deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] https://pkg.jenkins.io/debian-stable binary/" | sudo tee /etc/apt/sources.list.d/jenkins.list > /dev/null
     suppose if the 7,8 cmds are not working then first clean it using the follwing cmds and then try with 09,10 cmds

	sudo rm -f /etc/apt/sources.list.d/jenkins.list
	sudo apt clean

9. wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -
10. echo 'deb https://pkg.jenkins.io/debian-stable binary/' | sudo tee -a /etc/apt/sources.list.d/jenkins.list

11. sudo apt update
12. sudo apt install jenkins
